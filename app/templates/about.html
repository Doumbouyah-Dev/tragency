About Task Manager

Task Manager is a productivity tool built to help users create, manage, and track their daily tasks. Whether you're a student, professional, or entrepreneur, staying organized is key to success.

Features

  • User Registration and Login
  • Create, View, and Delete Tasks
  • Commenting and Notifications
  • Mobile and Desktop Friendly UI

Built With

  • Python & Flask
  • SQLAlchemy ORM
  • JWT Authentication
  • Bootstrap 5
